cache_disk

 cache_disk (base_file, rm_cache=False)

Decorator to cache function output to disk

import time
from tempfile import tempdir
cp = Path(tempdir) / "test_cache"
@cache_disk(cp)
def slow_add(a,b):
    time.sleep(1)
    return a + b

this time is the first time so not from the cache

CPU times: user 4 µs, sys: 1 µs, total: 5 µs
Wall time: 13.8 µs
3

now is much faster beacuse of the cache

CPU times: user 3 µs, sys: 1 µs, total: 4 µs
Wall time: 4.77 µs
3

adding comments change the hash, so the function is still cached

@cache_disk(cp)
def slow_add(a,b):
    time.sleep(1)
    # this is a comment
    return a + b

Wall time: 4.53 µs
3
CPU times: user 3 µs, sys: 1 µs, total: 4 µs
Wall time: 5.25 µs
